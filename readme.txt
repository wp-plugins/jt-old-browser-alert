== Description ==

오래된 브라우저[기본값: 익스플로러7 이하]로 접속할 시 사이트 상단에 
최신 브라우저로의 업데이트를 권하며 업데이트 경로를 안내합니다.

하위 브라우저 제한의 변경은
'플러그인' -> '편집기' -> '편집할 플러그인 선택' -> 'JT old browser alert' 선택 -> 플러그인 문서 내용 중
'[if lte IE 7]' 부분을 수정합니다.

예)
[if lte IE 7] 익스플로러7 이하, 
[if lte IE 6] 익스플로러6 이하

== Installation ==

1. 다운로드 받은 플러그인을 FTP를 통해 'Plugins' 폴더로 업로드 하거나, 
   워드프레스 관리자-> 플러그인 탭 에서 'jt-old-browser-alert' 검색 후 설치 
2. 'JT old browser alert' 플러그인 활성화
